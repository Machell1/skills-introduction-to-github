What is Perplexity AI?

Perplexity is an AI search engine that provides cited, sourced answers to your questions. It combines web search with AI analysis to deliver accurate, up-to-date information with references.

Key Features

  • Real-time web search
  • Follow-up questions
  • Collections (saved research)
  • File upload analysis
  • Focus modes
