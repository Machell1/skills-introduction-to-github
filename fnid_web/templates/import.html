Import Instructions
Supported File Types
  • .xlsx (Excel) - Spreadsheet with column headers matching database fields
  • .docx (Word) - Structured document with tables or labelled sections
Excel Import Guidelines
  • First row must contain column headers
  • Column names should match or closely approximate database field names
  • Dates should be in YYYY-MM-DD format
  • Use exact values from controlled dropdowns where possible

Important Notes
  • Validation: Imported records will be created with Draft status and must be reviewed before submission.
  • Duplicates: The system will generate new IDs for all imported records. Check for duplicates after import.
  • Audit Trail: All imports are logged in the audit trail with the importing officer's credentials.
  • Processing: After upload, use the CLI data import tool to process the file into the database.
